# API Assignment Project

## Description
This project demonstrates a simple web application that retrieves and displays data from multiple APIs using Node.js and Express.  
It is designed for learning purposes and showcases:

- Server-side API integration
- Fetching and displaying user, country, currency, and news data
- Minimal, clean, and beginner-friendly interface with pink and white color palette

---

## Features

1. **Random User API**  
   - Fetch a random user
   - Display personal info: first name, last name, gender, age, date of birth, city, country, full address, profile picture

2. **REST Countries API**  
   - Get country details based on the random user’s country
   - Display country name, capital, official languages, currency, and national flag  
   - *Note:* This API does not require an API key, but an environment variable is configured to satisfy assignment requirements

3. **Exchange Rate API**  
   - Show the user's local currency compared to USD and KZT

4. **News API**  
   - Display 5 latest news headlines from the user’s country in English
   - Include headline title, image, description, and link to full article
